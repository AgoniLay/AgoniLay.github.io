Cascading Style Sheets 层叠样式表（级联样式表）

好处

  • 降低耦合度（解耦）
  • 让分工协作更容易
  • 提高开发效率

使用方式

内联样式

在标签内使用 style 属性指定 css 代码

适用于属性较少时

内部样式

内嵌式：在 head 标签内，定义 style 标签，style 标签的标签体内容就是 css 代码

外部样式

a.css 文件

链接式 (推荐使用)

在 head 标签内，定义 link 标签，引入外部的资源文件

导入式

使用 @import 在 style 标签内导入，必须在标签内最前面

引号可以是单引号也可以是双引号，也可以省略引号

link 和 @import 的区别：

  • link 是 html 标签；而 @import 是 css 提供的
  • link 在页面加载时加载；@import 在页面加载完再加载
  • @import IE5 以上才可识别；link 无兼容性问题
  • link 样式的权重高于 @import 的权重

语法

  • 选择器：筛选具有相似特征的元素
  • 每一对属性需要使用 ; 隔开，最后一对属性可以不加

注释语法

在 css 中注释使用 \* 注释内容 *\

属性选择器

选择元素名称，属性名=属性值的元素

  • 元素名称[属性名] {} ：选择带有该属性的元素
  • 元素名称[属性名="属性值"] {} ：该属性的值为该属性值的 （常用）
  • 元素名称[属性名^="xxx"] {} ：属性值为 xxx 开头的
  • 元素名称[属性名$="xxx"] {} ：属性值为 xxx 结尾的
  • 元素名称[属性名*="xxx"] {} ：属性值的任意位置带有 xxx （常用）
  • 元素名称[属性名~="xxx"] {} ：属性值是一个词列表，并且以空格隔开，其中词列表中包含了 xxx
  • 元素名称[属性名|="xxx"] {} ：特定属性选择器，该属性值是 value 或者以 value- 开头的值（如 zh-cn）

伪类选择器

选择一些元素具有的状态

语法： E:pseudo-class {property: value;}

动态伪类
  • :link ：初始化的状态
  • :visited ：被访问过的状态
  • :hover ：鼠标悬浮状态（常用）
  • :active ：正在访问状态

注意顺序： link -> visited -> hover -> active 不可以调换

常用于自定义超链接样式

  • :focus ：聚焦的状态
结构伪类

  • :first-child ：首个子元素

  • :last-child ：最后一个子元素

  • :nth-child(n) ：

    • even === 2n –> 偶数项
    • odd === 2n-1 –> 奇数项
    • 只能是字母 n，其他的没有效果

  • :nth-last-child() ：从最后一个开始数，用法与 nth-child() 相同

  • :nth-of-type() ：用法类似 :nth-child()，但只计算选择器中指定的哪个元素，指定了子元素的类型

  • :nth-last-of-type() ：从最后一个开始计算

  • :first-of-type() :last-of-type() ：类似于 :first-child() 和 :last-child()

  • :only-child() ：表示一个元素是它的父元素的唯一一个子元素

  • :only-of-type() ：表示一个元素包含很多子元素，只有一种子元素是唯一的，那么就可以选中它

  • :empty() ：用来选择没有任何内容的元素，一个空格也没有

注意：IE6~8 和 FF3 及其以下版本不支持以上的结构伪类选择器

否定伪类
  • :not() 否定选择器，即排除或过滤掉特定元素。

注意：IE6~8 及其以下版本不支持否定伪类选择器

状态伪类

UI 元素（User Interface 用户界面）一般是指包含在 form 内的表单元素，其状态一般包括：可用不可用、选中未选中、获得或失去焦点、锁定、待机等

  • :enabled() 表示匹配指定范围内所有可用 UI 元素，匹配文本框，不匹配按钮
  • :disabled() 表示匹配指定范围内所有不可用 UI 元素，匹配按钮，不匹配文本框
  • :checked() 表示匹配指定范围内所有可用 UI 元素，匹配单选按钮

注意：IE6~8 及其以下版本不支持状态伪类选择器

目标伪类
  • :target ：匹配所有指定目标元素中被相关 URL 指向的（比如锚点链接指向）

注意：IE6~8 及其以下版本不支持目标伪类选择器

伪元素选择器

语法： 元素::状态 {}

  • ::first-letter ：选择第一个字
  • ::first-line ：选择第一行
  • ::selection ：选中文字时改变样式
  • ::before {content: "xxx"} ：在内部前面插入 xxx
  • ::before {after: "xxx"} ：在内部后面插入 xxx

属性

单位

绝对单位
  • in（inch，英寸）
  • cm（centimerter，厘米）
  • mm（millimeter，毫米）
  • pt（point，印刷的点数）
  • pc（pica，1pc == 12pt）
  • 另外，xx-small | x-small | small | medium | large | x-large | xx-large 这些关键词也是绝对单位
相对单位

  • px 像素

    • 根据屏幕的像素点的尺寸变化而变化，不同分辨率的屏幕所显示的像素字体大小是不同的
    • 屏幕分辨率越大，相同像素字体就显得越小

  • em 相对于父辈字体的大小来定义字体大小，1em 即是 父级一个字体大小

  • ex 相对于父辈字体的 x 高度来定义字体大小

    • ex 单位大小既取决于字体的大小，也取决于字体类型
    • 在固定大小的情况下，实际的 x 高度随字体类型不同而不同

  • % 以百分比的形式定义字体大小，于 em 效果相同，相对于父辈字体大小来定义

  • larger 和 smaller 将以父元素的字体大小为参考进行换算

颜色表示

  • 颜色名：使用颜色名，如：red，pink，skyblue
  • 十六进制：0~F 如： #fff，#6E6E6E，#000000
  • RGB：rgb(0255, 0255, 0~255)，其中也可以是百分比值

CSS3 新支持以下三种颜色表示法：

  • RGBA：在 RGB 颜色的基础上增加了 Alpha 通道，起到定义半透明的颜色，取值 0~1，越低越透明

  • HSL：色相(Hue)、饱和度(Saturation) 和 亮度(Lightness) 表示颜色

  • HSLA：和 RGBA 同理，添加透明通道

渐变

Webkit 渐变
  • type：定义渐变类型，包括线性渐变（linear）和 径向渐变（radial）
  • point：定义渐变起始点和结束点坐标，即开始应用渐变的 x 轴 和 y 轴 坐标，以及结束渐变的坐标，(0, 0)、(left, top)、top、bottom、left、right
  • radius：当定义径向渐变时，用来设置径向渐变的长度，为一个数值
  • stop：定义渐变色和步长
    • from(colorvalue)：开始的颜色
    • to(colorvalue)：结束的颜色
    • color-stop(value, colorvalue)：颜色步长、包含两个参数值，第一个为数值，第二个为颜色值
Gecko 渐变
线性渐变
  • point：定义渐变起始点
  • angle：定义直线渐变的角度
  • stop：定义步长，用法与 Webkit 的 color-stop() 函数相似
径向渐变
  • point：定义渐变起始点
  • angle：定义渐变的角度
  • shape：定义径向渐变的形状
  • size：定义圆半径，或者椭圆的轴长度
  • stop：定义步长，用法与 Webkit 的 color-stop() 函数相似
W3C 渐变

沿袭 Gecko 引擎的渐变设计方法，语法与用法也基本相同

字体样式

font-size 字体大小
  • 前面的表示绝对字体尺寸
  • larger 和 smaller 表示相对的大小，根据父对象中字体尺寸进行相对增大和缩小处理
  • length 可以是百分数、浮点数或单位标识符组成的长度值，不可谓负值
font-family 字体
  • 各种字体之间用 , 分隔开
  • 中文字体需要加英文引号，英文字体一般不需要加引号
  • 如果字体中包含 (空格) # $ 等符号，则必须加引号
  • Unicode 编码表示字体：
    • \5B8B\4F53 == SimSun –> 宋体
    • \5FAE\8F6F\96C5\9ED1 == Microsoft YaHei –> 微软雅黑
font-weight 字体粗细
  • 可以使用数字，可选数值为 100 的整数倍， 100 ~ 900，越大越粗
  • normal == 400；bold === 700 < 较为常用
  • bolder：相对于 normal 字体较粗；lighter：相对于 normal 字体较细
font-style 字体样式
  • normal：普通
  • italic：斜体
  • oblique：倾斜的字体

italic 和 oblique 只能在英文等西方文字中有效

color 字体颜色

使用相应颜色表示法即可

text-decoration 字体划线效果
  • normal：默认值
  • none：取消划线效果
  • underline：下划线效果
  • blink：闪烁效果
  • line-through：贯穿线效果（即中划线）
  • overline：上划线效果

可以多个值混合使用，如：text-decoration: line-through overline underline; 达到显示上中下三条划线效果

常用：取消 a 链接 默认下划线样式：text-decoration: none;

font-variant 定义小型大写字母
  • normal：默认值，即正常的字体
  • snall-caps：小型的大写字母字体

仅支持以英文为代表的西文字体

text-transform 单词大小写
  • none：默认值，无转换发生
  • capitalize：将每个单词的第一个字母转换成大写，其余无转换
  • uppercase：所有字母转换为大写
  • lowercase：所有字母转换为小写
@font-face 自定义字体类型

能够加载服务器端的字体文件

文本样式

text-align 水平对齐 *
  • left：左对齐，默认值
  • rigth：右对齐
  • center：居中对齐
  • justify：两端对齐

CSS3 新增四个属性：

  • start | end | match-parent | justify-all
  • 浏览器支持不是很好
vertical-align 垂直对齐 *
  • auto：根据 layout-flow 的值对齐对象内容
  • baseline：默认值，表示将支持 valign 特性的对象内容与基线对齐
  • sub：表示垂直对齐文本的下标
  • super：表示垂直对齐文本的上标
  • top：对象的内容与对象顶端对齐
  • text-top：对象的文本与对象顶端对齐
  • middle：对象的内容与对象中部对齐
  • bottom：对象的内容与对象底端对齐
  • text-bottom：对象的文本与对象底端对齐
  • length：表示由浮点数字和单位标识符组成的长度值或百分数，可为负数，定义由基线(0,0%)算起的偏移量
letter-spacing 字间距

取值为长度值

word-spacing 词间距

取值为长度值，默认值为 normal，表示默认间隔，以空格作为识别的分界线

line-height 行高 *
  • normal：默认值，一般为 1.2em
  • length 为长度值，允许为负值，可以不写单位，灵活继承父级单位
  • 一般行高的最佳设置范围为 1.2em~1.8em
  • 可以遵循字体越大，行高越小的原则来定义
text-indent 首行缩进
  • length 为长度值，允许为负值
  • 可以巧用：将不想显示的文本缩进出窗口外 text-indent: -9999px;
  • 建议以 em 为设置单位
text-shadow 文字阴影
  • none：无阴影
  • 前两个参数必须写，后两个可以省略
  • 前两个参数，正值偏右或偏下，负值相反
  • 阴影模糊值是一个长度值，不允许为负值
  • 模糊距离越大越虚
  • 颜色设置可以在调换到任意位置
  • 如未设置颜色，则继承自 color 属性的颜色
  • 设置多个阴影效果用 , 分隔
  • 可以使用在 :first-letter 和 :first-line 伪元素上
  • 在左上和右下各补一个 1 像素错位的补色阴影，可以起到外凸内凹的效果
text-overflow 溢出文本
  • clip：不显示省略标记 … ，简单的裁切
  • ellipsis：当对象文本溢出时显示省略标记 … ，插入位置是最后一个字符
  • ellipsis-word：当对象文本溢出时显示省略标记 … ，插入位置是最后一个词

要实现已出现是省略号的效果，还需要设置两个属性：

word-wrap 文本换行
  • normal：表示控制连续文本换行
  • break-word：内容将在边界内换行
content 添加动态内容
  • normal：默认值
  • string：插入文本内容
  • attr()：插入元素的属性值
  • uri()：插入一个外部资源，如图像、音频、视频或浏览器支持的其他任何资源
  • counter()：计数器，用于插入排序标识
  • none：无任何内容

一般和 ::before 一起使用，也可以单独使用

表格

border-spacing 分离单元格间距
  • 可以为一个值 或 两个值
  • 两个值时，第一个值表示行间距，第二个值表示列间距
empty-cells 控制空单元格显示
  • show：显示空单元格
  • hide：隐藏空单元格
  • 空单元格：指没有可视内容的单元格
    • 如果单元格的 visibility 属性值为 hidden，即便单元格包含内容，也认为是空单元格
    • &nbsp; 和 其他空白字符为可视内容
    • ASCII 字符中的 回车符(“\0D”)、换行符(“\0A”)、Tab 键(“\09”)和空格键(“\20”)表示不可视内容

如果表格行中所有单元格的 empty-cells 属性都为 hide，且都不包含任何可视内容，那么整行就等于设置了 display: none

caption-side 表格标题位置

定义标题的显示位置

  • top（表格上面） | bottom（表格底部） | left | right
border-collapse 合并单元格边框

常用显示类型都可以划分为 block 和 inline 两种类型

float 浮动显示
  • left：元素向左浮动
  • right：元素向右浮动
  • none：默认值，表示消除浮动
特性
  1. 浮动元素以块状显示，可以定义 height 和 width
  2. 浮动元素与流动元素可以混合使用，不会重叠，都遵循向上后下的显示规则，都受到文档流影响
  3. 仅能改变水平显示方式，不能改变垂直显示方式
  4. 浮动元素可以并列显示，如果包含框宽度不够，则会错行显示
clear 清除浮动

不是清除浮动元素，而是清除本身，即不允许当前元素与浮动元素并列显示

  • left：清除左边的浮动元素，如果左边存在浮动元素，则当前元素会换行显示
  • right：清除右边的浮动元素，如果右边存在浮动元素，则当前元素会换行显示
  • both：清除左右两边的浮动元素，不管哪边存在浮动对象，当前元素都会换行显示
  • none：默认值，允许两边都可以存在浮动元素，当前元素不会主动换行显示

clear 属性是专门针对 float 属性而设计的，依次仅能够对左右两侧浮动元素有效，对于非浮动元素是无效的

在结构最后添加：

position 定位显示
  • static：默认值，表示不定位，元素遵循 HTML 默认的流动模型
  • relative：表示相对定位，通过 top bottom left right 属性确定元素在正常文档流中的偏移位置，发生偏移后元素对象原空间保留不变
  • absolute：表示绝对定位，叫元素从文档流中拖出来，然后使用 top bottom left right 属性相对于其最接近的一个具有定位属性的父定位包含框进行绝对定位
    • 如果不存在有定位的上级元素，则相对于浏览器窗口定位，层叠顺序通过 z-index 属性来定义
  • fixed：表示固定定位、粘性定位，相对于视图本身固定，不会随浏览器窗口的滚动而变化，该元素会始终位于浏览器窗口内视图的某个位置，通过 top bottom left right 属性设置位置，不会受文档流动影响，与 background-attachment: fixed 属性功能相同

绝对定位元素以块状显示

设置相对定位元素在可视区之外，不会出现滚动条，而绝对元素会

一般来说：子绝（absolute）父相（relative）

z-index 定位层叠

通过 z-index 属性来确定定位元素的层叠等级

  • 只有在元素的 position 属性取值为 relation absolute fixed 时才可以使用
  • z-index 属性值越大，层叠级别就越高，如果属性值相同，则根据结构顺序层叠

  • IE 不支持 fixed 使用 z-index

  • 引擎私有属性格式同 columns

  • normal：根据浏览器默认设置进行解析，一般为 1em

  • color：定义列边框的颜色，功能与 column-rule-color 属性相同

  • flex-end：伸缩项目在侧轴起点边的外边距紧靠住该行在侧轴终点的边

  • flex-end：各行向伸缩容器的终点位置堆叠

  • space-around：各行在伸缩容器中平均分布，在两边各有一半的空间

决定伸缩容器剩余空间按比例应扩展多少空间

默认值为 1，负值也可以生效，与 flex-grow 功能相反

flex-basis 伸缩比率

默认值为 auto，负值不合法

overflow-x overflow-y 溢出处理
  • visiable：默认值，不剪切内容，也不添加滚动条，元素将会被剪切为包含对象的窗口大小，且 clip 属性设置将失效
  • auto：在需要时剪切内容并添加滚动条，为 body 和 textarea 的默认值
  • hidden：不显示超出元素尺寸的内容
  • scroll：当内容超出元素尺寸，则 overflow-x 显示为横向滚动条，overflow-y 显示为纵向滚动条
  • no-display：当内容超出元素尺寸，则不显示元素，此时类似添加了 display:none 声明。仅作为交流使用
  • no-content：当内容超出元素尺寸，则不显示内容，此时类似添加了 visibility:hidden 声明。仅作为交流使用

使用时应同时设置 overflow-x 和 overflow-y 的属性值，不要出现一个为 hidden，而另一个为 visiable 的情况；

还要避免编写依赖指定值为 visiable 的 overflow-x 和 overflow-y 属性的计算值的代码

2D 变换
rotate() 定义旋转
scale() 定义缩放
  • 两个参数表示定义宽和高缩放比例，第二个参数省略表示两值相同
  • number 参数值可以是正数、负数和小数
    • 正数值基于指定的宽度和高度将放大元素
    • 负数表示翻转元素（如文字反转），然后再缩放元素
    • 使用小于一的正数可以缩小元素

  • 分别表示 x 轴 和 y 轴 相对于原位置的偏移距离
  • 省略第二个参数则第二个参数默认为 0

  • 可以使用 translateX() 和 translateY() 分别设置

  • 以 x 轴和 或 y 轴 为轴旋转，与 rotate() 不同，这个会改变形状
  • 可以使用 skewX() 和 skewY() 分别设置

transition 设置过渡动画
transition-property 设置过渡属性
  • none：表示没有元素
  • all：默认值，表示针对所有元素，包括 :before 和 :after 伪元素
  • IDENT：指定 CSS 属性列表，几乎所有色彩、大小或位置等相关的 CSS 属性，都可以应用过渡
transition-duration 设置过渡时间

初始值为 0，适用于所有元素，包括 :before 和 :after 伪元素

transition-delay 设置延迟时间
  • 初始值为 0，适用于所有元素，包括 :before 和 :after 伪元素
  • 设置时间可以为正整数、负整数和 0
  • 为 0 时 必须设置单位是 s 或者 ms
  • 为负数时，过渡的动作会从该时间点开始显示，之前的动作被截断
  • 为正数时，过渡的动作会延迟触发
transition-timing-function 设置过渡动画类型
  • ease：平滑过渡，等同于 cubic-bezier(0.25, 0.1, 0.25, 1.0) 函数
  • linear：线性过渡，等同于 cubic-bezier(0.0, 0.0, 1.0, 1.0) 函数
  • ease-in：由慢到快，等同于 cubic-bezier(0.42, 0, 1.0, 1.0) 函数
  • ease-out：由快到慢，等同于 cubic-bezier(0, 0, 0.58, 1.0) 函数
  • ease-in-out：由慢到快再到慢，等同于 cubic-bezier(0.42, 0, 0.58, 1.0) 函数
  • cubic-bezier：特殊的立方贝塞尔曲线效果

功能和 transition 相同，区别在于：

  • transition 只能指定属性的开始值和结束值，及其之间的过渡效果
  • animation 可以定义多个关键帧以及定义每个关键帧中元素的属性值来实现更为复杂的动画效果

兼容性：

  • -webkit-animation、-moz-animation、-o-animation
  • IE6~9 不支持 animation 属性
@keyframes 设置关键帧
  • animationname：定义动画名称
  • keyframes-selector：定义帧的事件未知，也就是动画时长的百分比
    • 可取值：0%~100%、from（等价于 0%）、to（等价于 100%）
  • css-styles：表示一个或多个合法的 CSS 样式属性
  • 为了避免有兼容性的问题，应始终定义 0% 和 100% 位置帧
animation-name 定义动画名称

初始值为 none，定义一个适用的动画列表

animation-duration 定义动画时间
  • 定义播放时间
  • 默认属性值为 0
  • 为负值时，被视为 0
animation-timing-function 设置过渡动画类型

初始值为 ease，参数值和 transition-timing-function 相同

animation-delay 定义延迟时间

允许一个动画开始执行一段时间后才被应用

animation-iteration-count 定义播放次数
  • 默认值为 1，动画只播放一次
  • infinite 表示无限次，即 CSS 动画永远重复
  • 取值为非整数，将导致动画结束一个周期的一部分
  • 值为负数，则在交替周期内反向播放动画
animation-direction 定义播放方向
  • normal：默认值，每次循环都是向前播放
  • alternate：表示第偶数次向前播放，第奇数次项反方向播放
animation-play-state 定义播放状态

  • running：初始值，定义动画正在播放

  • paused：定义动画已暂停

  • 可以在 JavaScript 中使用该属性

animation-fill-mode 定义播放外状态
  • none：初始值，不设置对象动画外的转态
  • forwards：设置对象状态为动画结束时的状态
  • backwards：设置对象状态为动画开始时的状态
  • both：设置对象状态为动画结束或开始的状态

层叠性

CSS 样式表的优先级

作者定义的样式 > 用户设置的样式 > 浏览器的默认样式 > HTML 的默认样式

特殊：

CSS2 中，用户设置的带 !important 的样式要优先于 作者声明的 !important 样式

CSS 样式的优先级

计算权值的方式判断优先级

  • 权值为 0 的：通配符选择器
  • 权值为 1 的：标签选择器、伪元素选择器、伪对象选择器
  • 权值为 10 的：类选择器、属性选择器
  • 权值为 100 的：ID 选择器

对于多个选择器组合而成的复合型选择器，分别计算后相加

  1. 分值越高优先级越高
  2. 若分值相同则根据位置关系来进行判断，越靠近对象的优先级越高

另外，!important 表示最大优先级，要放置在声明语句与分号之间

被继承的值具有特殊性，权值为 0

即不管父级样式的优先级多大，被子级元素继承时，该原始的优先级权重都为 0

例子：

  • 内联样式优先级更高
  • 在相同特殊性下，遵循就近原则
  • !important 有最大优先权，不论远近
    • 但继承的 !important 内的样式也还是 0

继承性
