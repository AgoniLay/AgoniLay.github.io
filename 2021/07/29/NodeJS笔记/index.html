一、NodeJS 简介

1.1 Node.js 是什么

Node.js 是 JavaScript 运行环境

可以直接通过浏览器访问 JavaScript 代码

1.2 能干什么

作用：让 JavaScript 成为与 PHP、Python 等平起平坐的语言

概括：前端脱离后端，直接通过 JS 写项目

1.3 下载

官网：http://nodejs.org

中文网址：http://nodejs.cn

学习社区：condjs.org

  • nvm 工具：实现 nodejs 任意版本的切换
  • npm 工具：下载 nodejs 所需模块（工具库）
  • nrm 工具：切换 npm 下载源

  • L loop 循环（后续所有的 JS 代码执行都会循环以上命令）
